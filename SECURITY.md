# Security Policy

Thank you for helping us keep Viqo secure!

## Reporting Security Issues

The security of our systems and user data is Inferviqo's top priority. We appreciate the work of security researchers acting in good faith in identifying and reporting potential vulnerabilities.

Please report validated security vulnerabilities by opening a [GitHub Security Advisory](https://github.com/inferviqo/viqo/security/advisories/new) or emailing **security@inferviqo.com**.
